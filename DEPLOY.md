# VIDEO SHUFFLER — веб-версия
## Деплой на Railway (бесплатно, 5 минут)

### Шаг 1 — GitHub
1. Зайди на https://github.com
2. Нажми **New repository** → назови `video-shuffler` → Create
3. Загрузи все файлы из этой папки (перетащи или через кнопку Add file)

### Шаг 2 — Railway
1. Зайди на https://railway.app
2. Нажми **Start a New Project**
3. Выбери **Deploy from GitHub repo**
4. Выбери репозиторий `video-shuffler`
5. Railway сам найдёт Dockerfile и запустит сборку (~3 минуты)

### Шаг 3 — Публичный URL
1. В Railway открой проект → вкладка **Settings**
2. Найди раздел **Networking** → нажми **Generate Domain**
3. Получишь ссылку вида `video-shuffler-xxx.up.railway.app`
4. Открывай с айфона — всё работает!

---

## Как пользоваться
1. Открой ссылку в Safari на айфоне
2. Нажми **Выбери или перетащи видео**
3. Выбери файл из Фото или Файлов
4. Настрой параметры (длина сегмента, FPS, разрешение)
5. Нажми **▶ ЗАПУСТИТЬ**
6. Жди — прогресс отображается в реальном времени
7. Нажми **⬇ СКАЧАТЬ ВИДЕО**

---

## Важно про бесплатный тариф Railway
- $5 кредитов в месяц — хватит на ~500 часов работы сервера
- Сервер засыпает после 30 минут бездействия (первый запрос разбудит за ~10 сек)
- Файлы хранятся временно — скачай сразу после обработки
